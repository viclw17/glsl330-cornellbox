# glsl-cornellbox

Render Cornell Box with Path Tracing in GLSL.

## Requirements

* C++17
* OpenGL 3.3
* CMake 3.12 or Higher

## Setup

```sh
git submodule update --init
```

## Build

```sh
mkdir build
cd build
cmake ..
make
```
